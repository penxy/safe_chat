\doxysubsection{详细描述}
Qt doesn\textquotesingle{}t support Q\+Object multiple inheritance. But for declaring signals in class, it should be inherit Q\+Object. To avoid this issue, interface can provide some pure virtual methods, which allow to connect to some signal.

This file provides macros to make signals declaring easly. With this macros signal-\/like method will be declared and implemented in one line each.
